Cybersecurity Projects

Explore my cybersecurity projects, where I tackle real-world security challenges, implement security measures, and perform security testing to identify vulnerabilities.

Project 1: Web Application Penetration Testing

This project involves testing the security of web applications through manual and automated penetration testing techniques to identify vulnerabilities such as SQL Injection, XSS, and CSRF.

Technologies Used: Kali Linux, Burp Suite, OWASP ZAP, Metasploit

Project 2: Network Security Monitoring

A network security monitoring system was built using open-source tools to continuously monitor network traffic for suspicious activity, conduct packet analysis, and implement intrusion detection systems.

Technologies Used: Snort, Wireshark, ELK Stack, Suricata

Project 3: Vulnerability Assessment & Mitigation

This project focused on identifying vulnerabilities in a corporate network through vulnerability scanners and then applying necessary patches and mitigations to reduce the risk of a potential attack.

Technologies Used: Nessus, OpenVAS, Nmap

Project 4: Secure File Transfer System

A secure file transfer system was designed to ensure safe and encrypted communication for sending sensitive information, using secure protocols such as SFTP and SSH.

Technologies Used: OpenSSH, SFTP, GPG, Linux
