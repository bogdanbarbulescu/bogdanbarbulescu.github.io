Top 25 Open Source Cyber Security Tools

Red Team Tools (Offensive)

  1. Metasploit Framework - Penetration Testing Framework
  2. Nmap - Network Mapper (Utility for Network Exploration)
  3. YARA - Pattern Matching for Malware Identification
  4. WireGuard - VPN (Virtual Private Network)
  5. TheHive-Cortex (extension of TheHive) - Security Orchestration, Automation, and Response (SOAR)
  6. Velociraptor - Forensic and Incident Response
  7. Kali Linux - Penetration Testing OS (Operating System)
  8. Phish.Report - Anti-Phishing and Email Security

Blue Team Tools (Defensive)

  1. Zeek (formerly known as Bro) - Network Security Monitoring
  2. ClamAV - Antivirus
  3. OpenVAS - Vulnerability Scanner
  4. Osquery - Endpoint Visibility and Monitoring
  5. Wazuh - XDR (Extended Detection and Response) and SIEM
  6. AlienVault OSSIM (now part of AT&T Cybersecurity) - SIEM and Threat Intelligence
  7. MISP (Malware Information Sharing Platform & Threat Sharing) - Threat Intelligence and Information Sharing
  8. Parrot Security OS - Security-oriented Operating System
  9. Elastic Stack (formerly ELK Stack) - Analytics and Log Management
  10. Arkime (formerly Moloch) - Packet Capture and Search
  11. OSSEC - Host-based Intrusion Detection System (HIDS)
  12. Suricata - Intrusion Detection and Prevention System (IDS/IPS)
  13. Graylog - Log Management and Analysis
  14. Trivy - DevOps and Infrastructure-as-Code (IaC) Vulnerability Scanner
  15. OpenEDR - Endpoint Detection and Response (EDR)
  16. OpenIAM - Identity and Access Management (IAM)
  17. pfSense - Security Appliance (Firewall/VPN/Router)
