Network Automation

This project demonstrates how automation can streamline network management and increase operational efficiency.

Project Overview

This project is aimed at automating network configuration tasks using various technologies such as Python, REST APIs, and network devices like FortiGate firewalls. It involves creating automated scripts to manage firewall policies, configure networking devices, and maintain network security.

Key Technologies Used

  • Python for scripting
  • REST APIs for device management
  • FortiGate firewalls
  • Automation frameworks
  • Version control using Git

Project Details

Through this project, I developed a system that automates common network management tasks. For example, using a Python script, network engineers can create and modify firewall policies based on input parameters. This reduces the need for manual intervention, minimizes errors, and speeds up configuration changes.
