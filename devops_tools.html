List on DevOps Tools

1. Version Control tools

  • GitHub

    • Github: one of the largest and most advanced development platforms in the world. Millions of developers and companies build, ship, and maintain their software on GitHub. Some of its’ salient features are:

    • Collaborative Coding
    • Automation / CI & CD
    • Security including additional features for enterprise customers
    • Project Management

  • Bitbucket

    • Bitbucket: a very popular platform, with over 10 million registered users. Although it is a platform for hosting code, it goes beyond just code management. Teams can plan projects, collaborate on code, test, and deploy from a single platform. Some of its features are:

    • Tighter Jira and Trello integration.
    • Integrated CI/CD to build, test and deploy.
    • Pull requests and approve code review more efficiently.
    • Keep your code secured in the Cloud with IP whitelisting and 2-step verification.

  • GitLab

    • GitLab: an all-in-one DevOps Tool for rapid software delivery. It enables teams to perform all tasks right from Planning to SCM to Delivery to Monitoring and Security. Following are a few of its features:

    • Single interface, single conversation thread, and single data store to manage projects effectively – Single source of truth.
    • CI/CD for robust, scalable, and end-to-end automation to work together efficiently – Continuous Everything.
    • Built-in functionality for Automated Security, Code Quality and Vulnerability management & with tight governance, your DevOps speed never slows down.
